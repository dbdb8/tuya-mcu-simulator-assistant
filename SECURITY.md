# Security Policy

Please do not disclose signing keys, cloud credentials, private Debugfiles, device identifiers, or exploitable vulnerabilities in a public issue.

Report security concerns privately through GitHub Security Advisories for this repository. Include affected versions, reproduction steps, and the expected impact. Do not include production secrets; use redacted examples.

Updater private keys are never stored in this repository. Release artifacts are accepted only when their updater signatures verify against the public key bundled with the application.
